## Overview

This release establishes the Hollow House Institute **Datasets Core**
as a governed, auditable, and extensible dataset catalog.

It introduces:
- Canonical dataset metadata specification
- Dataset-level schema enforcement
- License-governed release model
- Public dataset catalog

## Included Datasets

- Sample dataset (39 messages)
- Demonstrates structure, labeling, processing, and governance
- Intended for research and evaluation only

## Governance & Ethics

All datasets are released under the Hollow House Institute dataset license
with explicit prohibitions on exploitative, diagnostic, or coercive use.

## Next Releases

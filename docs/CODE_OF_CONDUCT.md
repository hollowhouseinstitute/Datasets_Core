# Code of Conduct

This project follows a community code of conduct to create a welcoming and respectful environment.

Standards
- Be kind, patient, and respectful.
- Assume best intentions, be constructive in feedback.
- No harassment, discriminatory or abusive language.

Reporting
- If you experience or witness unacceptable behavior, report it to the maintainers at: [use GitHub contact or maintainer email].
- For sensitive reports, see SECURITY.md for secure contact guidance.

Consequences
- Violations may result in warnings, removal from contributions, or further actions at maintainers' discretion.

Thank you for helping build a safe and inclusive community.
